# 💰 Money Management

Приложение для планирования личного бюджета с двойной валютной системой, динамическими категориями и отслеживанием расходов по месяцам.

## ✨ Возможности

### 📊 Планирование бюджета
- **Ввод дохода** — укажите вашу зарплату в USD
- **Цель сбережений** — установите желаемую сумму накоплений
- **Гибкое распределение** — настраиваемые проценты по категориям
- **Умная логика накоплений:**
  - 💚 **< 100%** — остаток автоматически идёт в сбережения (ускоренное достижение цели)
  - ⚠️ **> 100%** — перерасход вычитается из сбережений (замедленное достижение цели)

### 🏷️ Категории расходов
- **8 базовых категорий:** Сбережения, Еда, Одежда, Развлечения, Транспорт, ЖКХ, Здоровье, Прочее
- **Кастомные категории** — добавляйте свои категории с emoji, цветом и процентом
- **Управление категориями** — редактирование, удаление, пересчёт в реальном времени

### 💱 Двойная валютная система
- **USD** — основная валюта для ввода и расчётов
- **BYN/RUB** — вторичная валюта с автоматическим пересчётом
- **Живые курсы обмена**:
  - Национальный банк Республики Беларусь (BYN)
  - Центральный банк Российской Федерации (RUB)
- **Автоматическое обновление** курсов при загрузке приложения

### 📅 Трекер расходов
- **Ежемесячный учёт** — добавляйте расходы с привязкой к месяцу
- **Автоматическое планирование** — расходы создаются автоматически в начале месяца на основе распределения
- **План vs Факт** — сравнение запланированных и фактических расходов по категориям
- **Категоризация** — траты привязываются к любой категории (включая кастомные)
- **Визуализация** — прогресс-бары расходов по категориям с отображением отклонений
- **Статистика** — общий бюджет, потрачено, остаток, процент использования

### 📈 Визуализация
- **Интерактивная диаграмма** — кликабельные сегменты для подсветки категорий
- **Прогресс-бары** — наглядное отображение распределения бюджета
- **Цветовая индикация** — каждая категория имеет свой цвет
- **Адаптивный дизайн** — корректная работа на мобильных и десктопах

## 🚀 Быстрый старт

### Требования
- Node.js 18+
- pnpm (рекомендуется) или npm

### Установка

```bash
# Клонирование репозитория
git clone https://github.com/username/money-management.git
cd money-management

# Установка зависимостей
pnpm install

# Запуск dev-сервера
pnpm dev

# Или запуск с предварительной проверкой линтера
pnpm dev:safe
```

Приложение будет доступно по адресу `http://localhost:5173`

### Сборка для production

```bash
pnpm build
```

Собранные файлы будут в директории `dist/`

## 💾 Хранение данных

Все данные сохраняются в **localStorage**:

| Ключ | Описание |
|------|----------|
| `moneyManagementBudget` | Зарплата, цель сбережений, проценты категорий |
| `moneyManagementExpenses` | История расходов по месяцам (плановые и фактические) |
| `moneyManagementSettings` | Выбранная валюта (BYN/RUB) |
| `moneyManagementCustomCategories` | Кастомные категории |
| `moneyManagementLastBudgetedMonth` | Последний месяц с автопланированием |

## 🎨 Особенности реализации

### Умный расчёт бюджета
```typescript
// Если сумма процентов < 100%
const effectiveSavings = baseSavings + remainingAmount;
// Остаток ускоряет достижение цели

// Если сумма процентов > 100%
const adjustedSavings = baseSavings * (1 - overspendPercent / 100);
// Перерасход замедляет накопления
```

### Динамические категории
- Пользователь может добавлять любое количество категорий
- Каждая категория имеет уникальный ID, emoji, цвет и процент
- Кастомные категории полностью интегрированы в диаграмму и трекер расходов

### Валютная конвертация
- Асинхронная загрузка курсов с обработкой ошибок
- Fallback-значения при недоступности API
- Форматирование сумм согласно локали

### Автоматическое планирование расходов
```typescript
// При наступлении нового месяца автоматически создаются записи расходов
// на основе заданного распределения бюджета
const budgetedExpenses = categories.map(cat => ({
  category: cat.key,
  amount: salary * (cat.percentage / 100),
  isBudgeted: true,      // пометка как запланированный
  isAutoGenerated: true  // создан автоматически
}));
```

- **Авто-генерация** — при открытии приложения в новом месяце
- **Ручное планирование** — кнопка "Запланировать расходы на месяц"
- **Сравнение** — отображение отклонений фактических расходов от плана
- **Гибкость** — плановые записи можно редактировать и удалять
